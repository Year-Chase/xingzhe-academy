## 16. V2.5 报名信息页

风格：克制、清晰、安全感，不过度医疗化，不做复杂动态表单。

页面文案：本信息仅用于本次活动组织与安全保障。

确认页：手机号脱敏展示（138****0000），身份证号脱敏展示（110***********123X）。

## 17. 活动群二维码交互

小程序活动详情页：不常驻展示大二维码。展示轻量入口"加入活动群"（紧凑卡片行+"查看"按钮或底部按钮），点击后弹出二维码弹窗。

报名成功后：允许自动弹出一次活动群二维码弹窗，用户关闭后详情页保留"加入活动群"入口。

签到码页：展示"加入活动群"入口按钮，点击后弹窗展示二维码。

弹窗内容：groupQrTitle, groupQrDescription, groupQrImageUrl，"长按识别二维码加入活动群"提示。

失败兜底：活动群二维码暂不可用，请联系活动组织者。
