ADMIN-PRD

行者学社 Admin 管理后台 PRD
1. 当前定位

Admin 是行者学社运营管理端，不是用户消费端。

当前版本：V2.3 本地运营框架版。

Admin 已从 P2 活动后台扩展为：

活动管理 + 订单中心 + 财务管理 + 发票管理 + CRM 用户运营 + QA 回归支撑。

Admin 的核心职责：

创建活动
发布活动
查看报名
查看订单
处理退款
查看财务
管理发票
查看用户运营画像
支撑线下活动履约
