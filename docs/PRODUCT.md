行者学社 PRODUCT
1. 产品定位

行者学社不是普通活动报名工具，也不是单纯后台管理系统。

行者学社是：

活动报名支付系统 + 精美证书传播系统 + 克制型用户运营系统。

它通过线下活动、证书、足迹、地图和同行关系，帮助用户记录行动经历，获得精神满足，并愿意分享给别人。

2. 核心飞轮

用户看到活动
→ 报名支付
→ 现场参加
→ 二维码核销
→ 自动生成活动证书
→ 用户分享证书 / 活动
→ 新用户通过分享进入活动
→ 新用户注册并绑定活动级邀请关系
→ 新用户报名支付
→ 后台统计用户、订单、邀请、证书、城市、收入
→ 运营基于数据做克制、温和、长期的用户运营。

8. 二维码与核销

规则：

支付成功生成报名二维码。
现场扫码核销。
核销后二维码变为已核销。
已核销不可重复核销。
后台未来可重置二维码。
核销后未来自动生成证书。
